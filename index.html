Frello - Final Project

A project management tool inspired by Trello, built with React, JavaScript, and SCSS for a dynamic UI. The backend, developed with Node.js, Express, and MongoDB, supports real-time updates using Socket.IO and secure user authentication with OAuth. Key features include task management, responsive design, and seamless collaboration.

Minesweeper

A classic Minesweeper game developed using JavaScript, HTML, and CSS. Players must uncover all non-mine cells without triggering any mines. The game includes features such as a timer, mine counter, and difficulty levels. The user interface is designed for an intuitive and engaging gameplay experience.

QuickNote

QuickNote is a note management app inspired by Google Keep. It allows users to create, edit, delete, and organize notes with ease. Built with React, JavaScript, and CSS, the app offers features like note archiving, search and filtering, and attachment of images or files. All data is managed using local storage, ensuring efficient offline access.
